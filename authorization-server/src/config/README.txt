Sample keystores
================

Two keystores with different keys which happens to have the same name.
The combination of two keystores is used to test JWT decoding when the key with the requested name is available,
but does not match the original key.

oauth.pkx
    Private + public key: oauth202206
    Password: tiger202206

oauth-second.pkx
    Private + public key: oauth202206
    Password: tiger202206
